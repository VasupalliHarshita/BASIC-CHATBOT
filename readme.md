Basic Chatbot (Python)

Project Overview

The Basic Chatbot is a simple rule-based chatbot developed using Python. It interacts with users by responding to predefined text inputs such as greetings, questions, and farewell messages. The chatbot uses conditional statements ("if-elif-else") to determine the appropriate response.

This project is designed for beginners to understand the fundamentals of Python programming, including functions, loops, user input, and decision-making.

---

Features

- Accepts text input from the user.
- Responds with predefined messages.
- Supports common greetings and simple questions.
- Continues the conversation until the user types "bye".
- Displays a farewell message before exiting.
- Handles unknown inputs with a default response.
- Console-based application with no external libraries required.

---

Technologies Used

- Python 3
- Built-in Python functions only

---

Python Concepts Used

- Functions
- Variables
- "if-elif-else" statements
- "while" loop
- User Input ("input()")
- Output ("print()")
- String methods ("lower()")
- "break" statement

Project Structure

Basic-Chatbot/
│

---

How the Chatbot Works

1. The program starts by displaying a welcome message.
2. The chatbot waits for the user to enter a message.
3. The input is converted to lowercase for easier comparison.
4. The chatbot checks the user's input using "if-elif-else".
5. It displays the appropriate predefined response.
6. The chatbot continues the conversation until the user enters "bye".
7. When "bye" is entered, the chatbot displays a goodbye message and exits.

---

Algorithm

1. Start the program.
2. Define the chatbot function.
3. Display the welcome message.
4. Start a "while" loop.
5. Read user input.
6. Convert the input to lowercase.
7. Compare the input using "if-elif-else".
8. Display the corresponding response.
9. If the user enters ""bye"", display a goodbye message and stop the loop.
10. End the program.

---

Sample Input

hello
how are you
what is your name
thank you
bye

Sample Output

===================================
      BASIC CHATBOT
===================================
Type 'bye' to exit.

You: hello
Bot: Hi!

You: how are you
Bot: I'm fine, thanks!

You: what is your name
Bot: I am a simple Python chatbot.

You: thank you
Bot: You're welcome!

You: bye
Bot: Goodbye! Have a nice day.

---

Time Complexity

- Time Complexity: O(1) per user input, since the chatbot checks a fixed number of conditions.
- Space Complexity: O(1), as only a few variables are used regardless of the number of conversations.

---

Learning Outcomes

After completing this project, you will learn how to:

- Create and use functions.
- Accept user input using "input()".
- Use "if-elif-else" statements for decision-making.
- Implement a continuous conversation using a "while" loop.
- Convert strings to lowercase using ".lower()".
- Exit a loop using the "break" statement.
- Build a simple interactive console application.

- Add a simple menu of available commands.
